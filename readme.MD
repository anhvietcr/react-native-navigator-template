# React Native Navigator Template
A React native template with Navigator v4

Preview  Application
------------------------------------------
[![GitHub release](screen-shot.PNG)](preview)

            ******************** STEP BY STEP ******************************
+ Requirement
```
yarn install
react-native link react-native-vector-icons
```

+ Android
```
react-native run-android
```

+ IOS
```
cd ios && pod install
react-native run-ios
```
